Get hands-on with Large Language Models in under 5 minutes! This guide will help you run your first chapter notebook using Google Colab's free GPU.

<Note>
No local installation required - everything runs in your browser with Google Colab's free T4 GPU (16GB VRAM).

## Run Your First Notebook

<Steps>
  <Step title="Open Chapter 1 in Google Colab">
    Click the badge below to open the first chapter notebook directly in Google Colab:

    [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/HandsOnLLM/Hands-On-Large-Language-Models/blob/main/chapter01/Chapter%201%20-%20Introduction%20to%20Language%20Models.ipynb)

  <Step title="Enable GPU Runtime">
    In Google Colab, ensure you're using a GPU runtime:

    1. Click **Runtime** in the menu bar
    2. Select **Change runtime type**
    3. Choose **T4 GPU** from the Hardware accelerator dropdown
    4. Click **Save**

    <Tip>
    The free T4 GPU provides 16GB of VRAM, which is sufficient for all examples in the book.
    </Tip>
  </Step>

  <Step title="Install Dependencies">
    Run the first code cell to install required packages:

    - Clicking the play button (▶) next to each cell, or
    - Pressing `Shift + Enter` to run a cell and move to the next one

    <Warning>

## Example: Your First Text Generation

Here's a taste of what you'll run in Chapter 1 - generating text with a pre-trained model:

```python
from transformers import pipeline

# Load a text generation pipeline
generator = pipeline('text-generation', model='gpt2')

# Generate text
prompt = "Large language models are"
output = generator(prompt, max_length=50, num_return_sequences=1)

print(output[0]['generated_text'])
```

<CodeGroup>
```python Simple Generation
from transformers import pipeline

# Create a text generation pipeline
generator = pipeline('text-generation', model='gpt2')

# Generate text from a prompt
result = generator("The future of AI is", max_length=30)
print(result[0]['generated_text'])
```

```python With Parameters
from transformers import pipeline

# Create pipeline with specific model
generator = pipeline('text-generation', 
                    model='gpt2',
                    device='cuda')  # Use GPU

# Generate with custom parameters
result = generator(
    "Large language models are",
    max_length=50,
    num_return_sequences=3,
    temperature=0.7,
    top_k=50
)

# Print all generated sequences
for i, sequence in enumerate(result):
    print(f"Sequence {i+1}: {sequence['generated_text']}\n")
```
</CodeGroup>
