## Key Topics Covered

### Tokenization
Explore how modern LLMs break down text into tokens, comparing different tokenization strategies used across popular models.

### Embeddings
Understand how raw tokens are transformed into rich vector representations that capture semantic meaning.

### Self-Attention
See animated visualizations of how attention mechanisms allow models to weigh the importance of different tokens when processing sequences.

### Recent Improvements
Learn about cutting-edge optimizations that make modern transformers faster and more efficient:

- **KV Cache**: Reduce redundant computation during text generation
- **Multi-Query Attention**: Share key and value projections across attention heads
- **Grouped Query Attention**: Balance between multi-head and multi-query attention
- **Sparse Attention**: Process longer sequences by attending to select tokens

### Hands-On Exploration
Work with the Hugging Face Transformers library to explore these concepts in practice.
