# Welcome to Hands-On Large Language Models

Welcome to the companion documentation for **Hands-On Large Language Models**, the comprehensive guide to understanding and implementing Large Language Models written by [Jay Alammar](https://www.linkedin.com/in/jalammar/) and [Maarten Grootendorst](https://www.linkedin.com/in/mgrootendorst/).

<Note>
This repository contains all code examples, notebooks, and resources from the book, playfully dubbed **"The Illustrated LLM Book"**.
</Note>

## What You'll Learn

Through **almost 300 custom-made figures** and hands-on code examples, you'll master the practical tools and concepts needed to work with Large Language Models today. The book covers 12 comprehensive chapters:

## Who This Is For

This book is perfect for:

- **Data scientists** looking to add LLMs to their toolkit
- **ML engineers** wanting to implement production-ready LLM applications
- **Researchers** exploring the latest advances in language models
- **Developers** building AI-powered applications
- **Students** learning about modern NLP and deep learning

While some machine learning background is helpful, the visual approach makes complex topics accessible to motivated learners at all levels.
</Note>

## How to Use This Documentation

This documentation serves as an interactive companion to the book:

1. **Get Started** - Set up your environment and run your first notebook
2. **Chapters** - Access all 12 chapter notebooks with detailed explanations
3. **Bonus Content** - Explore additional guides on cutting-edge topics
4. **API Reference** - Quick reference for key libraries and functions

<Tip>
All examples are designed to run on Google Colab with a free T4 GPU (16GB VRAM), making it easy to get started without local hardware requirements.
</Tip>
