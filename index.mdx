## Book Features

<Accordion title="12 Comprehensive Chapters">
  - **Fundamentals**: Introduction, tokens & embeddings, transformer architecture
  - **Applications**: Text classification, clustering & topic modeling
  - **Advanced**: Prompt engineering, text generation, semantic search & RAG, multimodal LLMs
  - **Development**: Creating embeddings, fine-tuning classification & generation models
</Accordion>

<Accordion title="Hands-On Code Examples">
  All examples are provided as Jupyter notebooks that run on Google Colab with free GPU access. Every chapter includes working code you can run immediately.
</Accordion>

<Accordion title="Visual Learning Approach">
  Almost 300 custom illustrations help you understand complex concepts intuitively. See how attention mechanisms, embeddings, and model architectures actually work.
</Accordion>

<Accordion title="Bonus Advanced Topics">
  Continue learning with in-depth guides on quantization, Mamba state space models, mixture of experts, stable diffusion, reasoning LLMs, DeepSeek-R1, and LLM agents.
</Accordion>
